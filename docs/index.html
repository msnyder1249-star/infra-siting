ERCOT Substation Capacity Index

A public-data screening tool for Texas substations that combines ERCOT market signals, ERCOT reference mappings, and substation voltage context to identify relatively promising large-load interconnection candidates.

Open Interactive Map Download Scored CSV Download Crosswalk CSV

What This Is

A screening index, not a formal interconnection study. Use it to shortlist substations for deeper utility and ERCOT diligence.

What It Includes

  • ERCOT bus and resource-node mappings
  • LMP, hub-price, and shadow-price signals
  • Normalized Texas substation inventory

Important Limitation

The map does not report exact MW headroom. True usable capacity still requires planning cases, facility ratings, and interconnection studies.
